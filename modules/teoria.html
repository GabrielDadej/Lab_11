Podstawy języka Kotlin

Naucz się podstaw programowania w Kotlinie

1. Czym jest Kotlin?

Kotlin to nowoczesny język programowania stworzony przez firmę JetBrains. Jest oficjalnym językiem do tworzenia aplikacji Android.

  • prosty składniowo,
  • bezpieczny,
  • szybki,
  • współpracuje z Javą.

2. Funkcja main()

Każdy program w Kotlinie rozpoczyna działanie od funkcji main().

fun main() { println("Hello Kotlin!") }

3. Zmienne

W Kotlinie używamy:

  • val — stała (nie można zmienić wartości)
  • var — zmienna (wartość można zmieniać)
val name = "Kotlin" var age = 20 age = 21

4. Typy danych

Najpopularniejsze typy danych:

  • Int — liczby całkowite
  • Double — liczby dziesiętne
  • String — tekst
  • Boolean — true / false
val number: Int = 10 val price: Double = 19.99 val text: String = "Hej" val isOnline: Boolean = true

5. Instrukcja if

Instrukcja warunkowa pozwala wykonywać kod zależnie od warunku.

val age = 18 if(age >= 18){ println("Pełnoletni") }else{ println("Niepełnoletni") }

6. Pętle

Pętle służą do powtarzania kodu.

for(i in 1..5){ println(i) }

7. Funkcje

Funkcje pozwalają organizować kod.

fun add(a: Int, b: Int): Int { return a + b }

8. Klasy i obiekty

Kotlin wspiera programowanie obiektowe.

class User(val name: String) fun main(){ val user = User("Adam") println(user.name) }
